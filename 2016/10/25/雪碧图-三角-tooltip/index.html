css Sprites雪碧图

CSS Sprites技术被国内一些人称为CSS雪碧图，就是把网页中一些背景图片整合到一张图片文件中，再利用CSS的background-image，background- repeat，background-position的组合进行背景定位，background-position可以用数字精确的定位出背景图片的位置。
如下图左边是京东页面的一个侧边栏，边栏中的图标则全在右边这张图里面，通过background-position的不同来使用各个图标。

优点：

  • 相对于当个小图标，它节省文件体积和服务请求次数。将所有零碎的网页背景图片整合到一起，这样做可以有效的减少http对图片的请求次数，而不需要加载多次加载零碎的背景图片，所以合理的利用好它可以有效的提高网页的加载速度。只需要加载一次图片，以后再需要加载这张图片时，可以从缓存中找到。

缺点：

  • 需预先确定每个小图标的大小
  • 图标大小的尺寸是固定的，在响应式页面中更改网页的大小时，会使图片变得模糊

绘制三角形

应用场景


三角形在网页中是个很常用的显示元素，这些三角形用图片都是可以达到一样的效果的，但当颜色和比例需要改动时，CSS只要简单的改变一下border的样式就可以了，图片则需要要重新替换，效果也没有那么容易调试，另外过多的图片也会对页面性能造成影响，所以还是推荐用CSS来实现。

步骤

1、css绘制三角形主要通过border属性来完成


2、将width、height设置为0


3、这样就可以根据需要，将不需要边框的颜色隐藏就可以得到我们想要的三角形


上边的 width 控制了三角形的高（50px），左右两边的 width 分别控制了三角形的底边长的两部分（加起来共 100px）。
4、此外我们可以留着上边框和左边框，就可以构造直角三角形

Tooltip

tootip就是我们平常在网页中看到的提示框,它的原理就是给一个边框加上一个三角形。

实心的三角形

线条型的三角形

有两种实现方式：
a. 再添加一个白色的三角形，然后通过定位去覆盖绿色的三角形的一部分。弊端：此时三角形的边框的宽度和矩形宽度可能不一致。


b.给定一个内部div，设置它的边框（如保留上边框和左边框），并让它旋转45度角，并设置与父元素的位置。弊端：这是一个css3属性，IE8不支持。
