tab切换

发表于 2016-12-21   |   分类于 JavaScript   |     |   阅读次数

效果


当点击tab1时，下面显示内容一；点击tab2时，显示内容2；点击tab3时，显示内容3。

思路

html结构如下：

  • 1、 默认的情况下，tab1与div1处于激活状态；
  • 2、对于tab,给它添加active，会给它设置背景色，对于div，给它添加active，会让它的display:block，其余div默认的display:none；
  • 3、 当一个tab被点击了，首先移除所有li的active类属性，然后给被点击的li添加类属性active，再移除所有div的active类属性，然后给被点击的li对应顺序的div添加类属性active；
  • 4、为了避免给所有li添加click事件（因为这样代码显得很冗杂），可以通过事件代理的方式将click事件添加到li父元素ul上面，然后通过event.target来捕获到底是点击的哪个li。
