2017年小计划

发表于 2017-01-04   |   分类于 生活   |     |   阅读次数

学习上：

1、学前端：还是喜欢Web前端，以后的就业方向应该就是这个了，并且今年下半学期要找工作了，所以今年最重要的还是继续学习前端相关知识。现列出上半学期主要学的技能与知识点如下：

  • jQuery
  • HTML5
  • CSS3
  • Node.js
  • Bootstrap

2、发小论文：争取2017年上学期把小论文发了，否则到时候又要找工作心里又有小论文的事情，压力会很大。所以多看论文，多做实验吧。虽然不想做，但为了毕业，还是要硬着头皮做啊。
3、多读书：2017年注定是忙碌的一年，但还是希望能多看点书，规定每个月至少4本书的阅读量吧，这样一年下来至少也能读50本书啦。
4、坚持练古筝：每次听到大师们弹的古筝曲，都觉得怎么可以这么美，所以还是要坚持每天练琴一个小时，争取2017年能达到5级的水平吧。这几天有点松懈了，打手！！！
5、学做烘焙：一直对做小饼干、小蛋糕很有兴趣，所以希望可以学习一下。
6、多尝试一些有趣的事情

生活上：

1、早睡早起：周间7：10分起床，8:30 必须在实验室开始学习，中午1个小时午觉，晚上11:30睡觉。周末可以睡会儿懒觉。
2、健康饮食：每天早上醒来喝一杯水，每天喝一杯酸奶、至少两杯柠檬水、吃点水果，少吃油腻、辛辣的食物。每周最多只能吃一次火锅、烤鱼、烤肉等辛辣油腻的食物，否则容易上火。（不过，在重庆，少吃麻辣食物真的有点难度啊！！）
3、洗脸护肤：每天认真洗脸护肤，每周2-3次面膜吧，学画淡妆。
4、保持房间整洁：寝室空间本身就小，所以没用的东西及时扔掉，每天晚上回寝室后轻微整理一下桌子，周末好好整理一下房间，每周的脏衣服不能推到下一周再洗。
5、每周买一束花：房间里有鲜花，心境会不一样。所以每周少买点零食，把节省出来的钱买花吧！
6、记录每天的消费：每个月都不知道钱花哪儿去了，所以还是记录一下吧。
7、可以出去旅游一次：具体去哪儿，还不知道，所以今年的压岁钱就少花点儿吧。

其他：

1、多关心父母，多给他们打电话，发微信。生日、大节记得买礼物！
2、和朋友相处，不要太计较，多站在别人的角度想问题。
3、不要太在意别人的看法，把更多的时间花在有价值、有意义的事情，少抱怨一点。
4、少看点无聊的综艺、视频，少玩点手机。
5、多思考，多总结。

2017年注定是忙碌的一年，拒绝拖延症，行动起来！！！
